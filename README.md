# Smart To-Do Manager

A modern and responsive task management web application built using **HTML, CSS, and JavaScript**. The application helps users organize tasks efficiently with priorities, due dates, progress tracking, search, filtering, theme switching, and recycle bin functionality.

## Features

### Task Management

* Add new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* Restore deleted tasks from Recycle Bin
* Permanently delete tasks

### Dashboard

* Total Tasks Counter
* Pending Tasks Counter
* Completed Tasks Counter
* Overdue Tasks Counter
* Dynamic Progress Bar

### Search & Filters

* Search tasks instantly
* Filter tasks by:

  * All Tasks
  * Pending Tasks
  * Completed Tasks

### Theme Support

* Dark Theme
* Light Theme
* Theme preference saved using Local Storage

### Data Persistence

* Tasks saved in Local Storage
* Data remains available after page refresh

### Responsive Design

* Mobile Friendly
* Tablet Friendly
* Desktop Friendly
* Adaptive Dashboard Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Font Awesome Icons
* Google Fonts (Poppins)
